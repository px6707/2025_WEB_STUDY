 # Generator 函数

Generator 函数是 ES6 引入的一种特殊函数，具有以下特点：

1. **基本语法**
   - 函数名前面有一个 `*` 号
   - 函数体内部使用 `yield` 表达式
   - 返回一个遍历器对象（Iterator）

2. **核心概念**
   - `yield`: 暂停执行的标记
   - `next()`: 恢复执行的方法
   - 可以通过 `next()` 传值
   - 具有 `return` 语句的效果

3. **主要特性**
   - 函数状态可以暂停
   - 可以多次返回
   - 支持双向通信
   - 提供迭代器接口

4. **常见用途**
   - 实现异步编程
   - 控制流管理
   - 创建数据流
   - 实现迭代器

5. **优势**
   - 简化异步编程
   - 提供更优雅的迭代方案
   - 更好的流程控制
   - 节省内存（惰性求值）
